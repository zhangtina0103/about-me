# Tina Zhang — Personal Site

React + Vite portfolio site with experience, resume, and projects.

## Local development

```bash
npm install
npm run dev
```

## Deploy to GitHub Pages

1. Push this repo to GitHub (e.g. `zhangtina0103/my-website-react`).
2. In the repo **Settings → Pages**, set **Source** to **GitHub Actions**.
3. Push to `main` — the workflow builds and deploys automatically.

Your site will be live at:

- **Project site:** `https://<username>.github.io/<repo-name>/`
- **User site:** rename the repo to `<username>.github.io` and change the workflow `VITE_BASE_PATH` to `/`

## Customize content

Edit `src/data/siteData.js` to update bio, experience, skills, and projects.
