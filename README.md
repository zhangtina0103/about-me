# Kristina Zhang — Personal Site

React + Vite portfolio site.

## Local development

```bash
npm install
npm run dev
```

Open http://localhost:5173

## GitHub Pages

Live URL: https://zhangtina0103.github.io/about-me/

**If the page is blank/white**, GitHub is serving source files instead of the built site:

1. Repo **Settings → Pages**
2. Under **Build and deployment → Source**, choose **GitHub Actions** (not “Deploy from a branch”)
3. Clear any **Custom domain** field unless you own a domain like `yoursite.io`
4. Go to **Actions** → **Deploy to GitHub Pages** → **Re-run all jobs**

After a green deploy, hard-refresh the site (Cmd+Shift+R).
